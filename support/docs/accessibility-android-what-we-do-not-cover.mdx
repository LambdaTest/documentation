## 3. What we do NOT cover (checklist + how to do – Android)

### 3.1 Time-based media (1.2.1–1.2.9)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 1.2.1–1.2.9 | Play in-app media; verify captions, transcripts, audio descriptions are present and accurate. |

### 3.2 Meaningful sequence, sensory, orientation, input purpose (1.3.2–1.3.5)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 1.3.2 Meaningful Sequence | With TalkBack, verify reading order preserves meaning. |
| 1.3.3 Sensory Characteristics | Verify instructions don't rely only on color/shape/location. |
| 1.3.4 Orientation | Rotate device; verify both orientations work. |
| 1.3.5 Identify Input Purpose | Verify autofill works for user info fields. |

### 3.3 Color, audio, resize, reflow (1.4.1, 1.4.2, 1.4.4, 1.4.10–1.4.13)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 1.4.1 Use of Color | Verify info not conveyed by color alone. |
| 1.4.2 Audio Control | Verify auto-play audio has pause control. |
| 1.4.4 Resize Text | Set font to largest; verify no truncation. |
| 1.4.10 Reflow | Verify content reflows without horizontal scroll. |

### 3.4 Keyboard/AT and no trap (2.1.1, 2.1.2)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 2.1.1 Keyboard | With TalkBack, complete all tasks via gestures. |
| 2.1.2 No Keyboard Trap | Verify focus can leave all components. |

### 3.5 Timing, pause/stop (2.2.1, 2.2.2)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 2.2.1 Timing Adjustable | Verify time limits can be extended. |
| 2.2.2 Pause, Stop, Hide | Verify auto-updating content can be paused. |

### 3.6 Flashes, animation (2.3.1, 2.3.3)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 2.3.1 Three Flashes | Verify no content flashes >3 times/second. |
| 2.3.3 Animation (AAA) | Verify animations respect Remove animations setting. |

### 3.7 Bypass, focus order, link purpose, focus visible (2.4.1, 2.4.3, 2.4.4, 2.4.7)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 2.4.1 Bypass Blocks | Verify skip-to-main exists for repeated nav. |
| 2.4.3 Focus Order | With TalkBack, verify logical focus order. |
| 2.4.4 Link Purpose | Verify button/link labels are clear. |
| 2.4.7 Focus Visible | Verify focus indicator is visible. |

### 3.8 Multiple ways, headings (2.4.5, 2.4.6)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 2.4.5 Multiple Ways | Verify 2+ ways to reach content. |
| 2.4.6 Headings and Labels | Verify headings describe content. |

### 3.9 Focus obscured, dragging, target size (2.4.11, 2.5.7, 2.5.8)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 2.4.11 Focus Not Obscured | Verify focused element visible under overlays. |
| 2.5.7 Dragging | Verify single-tap alternative to drag. |
| 2.5.8 Target Size | Verify 24dp minimum (48dp recommended). |

### 3.10 On focus, on input (3.2.1, 3.2.2)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 3.2.1 On Focus | Verify focus alone doesn't change context. |
| 3.2.2 On Input | Verify input alone doesn't submit unexpectedly. |

### 3.11 Consistent nav, identification, help (3.2.3, 3.2.4, 3.2.6)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 3.2.3–3.2.6 | Verify nav, components, help are consistent across screens. |

### 3.12 Errors, labels, redundant entry, auth (3.3.1–3.3.8)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 3.3.1 Error Identification | Verify errors announced by TalkBack. |
| 3.3.2 Labels | Verify all inputs have labels. |
| 3.3.7 Redundant Entry | Verify same info not re-requested. |
| 3.3.8 Accessible Auth | Verify CAPTCHA has alternative. |

### 3.13 Language (3.1.1, 3.1.2)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 3.1.1–3.1.2 | Verify language set for TalkBack pronunciation. |

### 3.14 Status messages (4.1.3)

| Checklist | How to do (manual test – Android) |
|-----------|------------------------------------|
| 4.1.3 | Verify status updates announced without focus move. |
