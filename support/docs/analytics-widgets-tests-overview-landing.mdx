# Analytics Widgets - Tests Overview

Get a comprehensive overview of your test analytics with <BrandName /> dashboard widgets.

<Note>
This page provides an overview of test analytics widgets available on the <BrandName /> platform.
</Note>

## Overview

The Tests Overview widgets provide key insights into your test execution metrics, including:

- Total tests executed
- Pass/fail rates
- Test duration trends
- Browser/platform coverage

## Available Widgets

- **Test Summary**: Quick snapshot of test results
- **Trend Analysis**: Historical test data visualization
- **Coverage Metrics**: Browser and OS coverage statistics
